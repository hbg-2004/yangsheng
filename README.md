# 养生生活 - 健康生活方式平台

专业的健康生活方式网站，提供减脂、睡眠改善、冥想、补剂、中老年健康和职场减压等内容。

## 🚀 快速开始

### 安装依赖

```sh
npm install
```

### 开发环境

```sh
npm run dev
```

访问 http://localhost:5173 查看网站

### 生产构建

```sh
npm run build
```

### 类型检查

```sh
npm run type-check
```

## ☁️ 部署到 Vercel

### 方法一：使用 Vercel CLI

1. 安装 Vercel CLI：
```sh
npm i -g vercel
```

2. 登录 Vercel：
```sh
vercel login
```

3. 部署项目：
```sh
vercel
```

4. 生产环境部署：
```sh
vercel --prod
```

### 方法二：使用 GitHub 部署

1. 将项目推送到 GitHub 仓库
2. 访问 [vercel.com](https://vercel.com)
3. 点击 "New Project"
4. 选择你的 GitHub 仓库
5. Vercel 会自动检测 Vue 项目并配置
6. 点击 "Deploy"
